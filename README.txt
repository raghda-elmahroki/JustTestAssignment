###########################JustTestAssignment Readme #########################################
This project is created using dotnetcore3.1 , nunit ,specflow ,C# ,selenium and VisualStudion2019

####How to run the tests######
Install dotnetcore3.1 
clone the project from github
navigate to the project folder and open command line
run command "dotnet test JustTestAssignment.csproj"

The tests should start running
